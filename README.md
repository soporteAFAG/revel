
# Revel MVP

Bienvenido a Revel, una Progressive Web App (PWA) minimalista para comenzar un viaje de autodescubrimiento emocional.

Este proyecto fue generado con React, TypeScript y estilizado con Tailwind CSS.

## Cómo ejecutar el proyecto

Sigue estos pasos para levantar el proyecto en tu máquina local.

### Prerrequisitos

- [Node.js](https://nodejs.org/) (versión 18 o superior)
- [npm](https://www.npmjs.com/) (generalmente viene con Node.js)

### Pasos

1.  **Clona el repositorio**
    Si aún no lo has hecho, clona el repositorio a tu máquina local.

2.  **Instala las dependencias**
    Navega a la carpeta del proyecto en tu terminal y ejecuta:
    ```bash
    npm install
    ```

3.  **Inicia el servidor de desarrollo**
    Una vez que las dependencias estén instaladas, puedes iniciar la aplicación con:
    ```bash
    npm run dev
    ```
    Esto iniciará la aplicación en modo de desarrollo. Abre tu navegador y visita `http://localhost:5173` (la URL podría variar, revisa la salida de tu terminal) para ver la aplicación en funcionamiento.

### Scripts disponibles

- `npm run dev`: Inicia el servidor de desarrollo.
- `npm run build`: Compila la aplicación para producción.
- `npm run preview`: Sirve la aplicación compilada localmente para previsualizar la versión de producción.
